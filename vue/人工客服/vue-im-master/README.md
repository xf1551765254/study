# vue-im
一个基于Vue2.0的在线客服系统。包括服务端和客户端。

# Features
* 支持1客服对多用户
* 支持客户选择客服
* 输入框支持文本、图片、表情、文件传输
* 输入框支持粘贴图片、文本表情混合
